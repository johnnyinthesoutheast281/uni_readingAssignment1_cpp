// Chapter 10: Arrays //

Definition: Arrays are sequences of something of the same type together.

To declare an array, the syntax is : (type) (name of the array)[(number of elements)];

Ex: int arr[5];
        
To initialize the array, we can use the initialization list " {  } " with comma between elements

Ex: int arr1[6] = {1, 3, 5, 7, 9, 0};

The counter of C++ is from 0, so in order to access the n element, we use subscript [(index minus 1)]

Ex: int arr2[3] = {56, 34, 12};
    cout << arr2[2] << "\n"; // This will print out the 3rd element in the array arr2: 12 //

To change the value, for example, of the 1st element of the array arr2, we use the following syntax:

    arr2[0] = 1;
    cout << arr2[0] << "\n"; // This will print out 1 // 
    
    



// Chapter 11: Pointers //

Definition: Pointers are types that can hold the address of a specific object.

To declare a pointer pointing to a specific data type, the syntax is as follow: (data type)* (name of the pointer);

Ex: char* ptr; //declare pointer ptr//

To assign the address of the variable that the pointer will hold, here is an example: 

Ex: int a = 6;
    int* ptr;
    ptr = &a; // The reference symbol " & " means that we take the address of the variable a //
    
Or, it can be :

    int* ptr = &a;

for instance.

To get the value of the address that the pointer is holding: 

Ex: int a = 6;
    int* ptr = &a;
    cout << ptr << "\n"; // This will print out the address that the pointer ptr is holding //
    cout << *ptr << "\n"; // The value of the address the pointer ptr is holding //

To assign another value for a variable using pointer: 

Ex: int a = 6;
    int* ptr = &a;
    cout << a << "\n"; // Print out 6 //
    *ptr = 7;
    cout << a << "\n"; // Print out 7 //

To declare pointer does not point to any object, we use nullptr as follow: 

Ex: int *b = nullptr;





// Chapter 12: References //

Definition: Reference is an alias to an existing object in memory. References must be initialized.

Ex: int a = 1;
    int& b = a;

If we assign another value to either one of the variable, the other will change, too.

Ex: int a = 1;
    int& b = a;
    b = 5;
    cout << b << "\n"; //Print out 5//
    cout << a << "\n"; //Print out 5//
    
 Another concept is const-reference, a read-only alias to some object.
 
 Ex: int a = 1;
     const int& b = a;
 
 
 
 
 
 // Chapter 13: Introduction to strings //
 
To use the string type, we need to add <string>, which is a standard library.

    #include <string>
   
To concatenating strings, we can simply use the " + " operator:

Ex: #include <iostream>
    #include <string>
    using namespace std;
    
    int main(){
        string a = "Hi ";
        string b = "there!";
        string c = a + b;
        cout << c << "\n";     //This line will print out "Hi there!"//
        cout << a + b << "\n"; //This line is the same with the upper one//
        return 0;
    }

To access the ith element of a string, we can use the [] operator like we use it with arrays or with a built-in function at() :

Ex: string a = "There is a bug";
    char b = a[2];
    char c = a.at(2);
    cout << b << "\n"; //This line prints out "e"//
    cout << c << "\n"; //This line is the same//
    
To compare two strings, we can simply use the logical operators :

Ex: string a1 = "In the end";
    string a2 = "in the end";
    if (a1 != a2){
       cout << "Two different strings";
    else{
       cout << "Both are one";
    }
    //Prints out "Two different strings"//
    
Get the string input: 

Ex: string s;
    cin >>  s; //Type in "abcxyz"//
    cout << s; //Prints out "abcxyz"//
    
We can use pointer points to the string using c_str() built-in function since string is an array: 

Ex: string a = "asd";
    cout << a.c_str() << "\n"; //Prints out "asd"//
    cout << (void*) a.c_str(); //Prints out the address of string a//

To copy a part of a string, we can use the built-in function substr():

Ex: string a = "abc";
    string b = a.substr(1,2); //copy from 2nd element until get enough 2 characters//
    
Find a substring using find() function:

Ex: string s = "abcxyz";
    string b = "xy";
    string::size_type found = s.find(b);
    
    if (found != std::string::npos){  //// If find() cannot find the substring, it returns std::string::npos ////
        cout << "Found at : " << found;
    }
    else{
        cout << "Cant find";
    }
